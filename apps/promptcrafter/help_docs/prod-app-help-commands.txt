Starting & Building the app/docker container:

	docker compose -f docker-compose.prod.yml up -d --build
	docker compose -f docker-compose.prod.yml up -d

------------------------------------------------------------------------------------

Initialize the .env file if not already done:

	nano .env
	
------------------------------------------------------------------------------------

Stopping the app/docker container:

	docker compose -f docker-compose.prod.yml down

------------------------------------------------------------------------------------

other commands for clearing storage space:

	docker builder prune --all
	docker image prune -a
	docker volume prune
	docker network prune
	docker container prune

------------------------------------------------------------------------------------

Get into bash of the container app:

	docker exec -it promptcrafter-chat-db-1 bash
